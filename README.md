# 🌱 Умный полив на ESP8266

Автоматическая система полива комнатных растений с локальным управлением через веб-интерфейс, цветным LCD дисплеем, автополивом по расписанию и защитой от отсутствия воды.

## 🚀 Возможности

- **Автоматический полив** по расписанию (12:00, 17:00, 20:00) при влажности почвы ниже 80%
- **Ручной полив** через веб-интерфейс или длительным нажатием кнопки
- **Контроль наличия воды** — после 3 неудачных поливов (влажность не поднялась) система блокируется
- **Псевдо-спячка** — дисплей автоматически выключается через 3 минуты бездействия
- **Два слайда на LCD**:
  - Главный: время, дата, влажность почвы, режим (ON/OFF)
  - Погодный: температура, ощущаемая температура, влажность воздуха, ветер (фиксированные данные для Азнакаево)
- **Локальный веб-сервер** для управления через Wi-Fi (точка доступа ESP)
- **Синхронизация времени** с телефоном через веб-интерфейс

## 🔧 Аппаратная часть

| Компонент | Назначение | Пин |
|-----------|------------|-----|
| ESP8266 NodeMCU | Главный контроллер | — |
| LCD I2C 16×2 | Дисплей | SDA → D2, SCL → D1 |
| Кнопка (левая) | Пробуждение, полив (длинное нажатие) | D4 (INPUT_PULLUP, замыкает на GND) |
| Кнопка (правая) | Переключение слайдов (короткое), переключение режима (длинное) | D5 (INPUT_PULLUP, замыкает на GND) |
| Датчик влажности почвы | Измерение влажности (аналоговый) | A0 (340 = мокро, 1024 = сухо) |
| Реле JQC3F-05VDC-C | Управление помпой | D7 |
| Помпа | Подача воды | через реле |

## 🔌 Подключение

### LCD I2C
- VCC → 5V (VU)
- GND → GND
- SDA → D2
- SCL → D1

### Кнопки
- Один контакт → D4 или D5
- Другой контакт → GND
- В коде используется `INPUT_PULLUP`, внешние резисторы не нужны

### Датчик влажности
- VCC → 5V (VU)
- GND → GND
- A0 → A0

### Реле и помпа
- **Важно!** Реле управляется нестандартно из-за просадок питания при коммутации:
  - Включение: `pinMode(D7, OUTPUT); digitalWrite(D7, LOW);`
  - Выключение: `pinMode(D7, INPUT);` (перевод в Hi‑Z)
- Помпа питается от отдельного источника (аккумулятор), реле коммутирует её цепь
- ESP8266 и реле питаются от USB (5V с пина VU)

## 🎮 Управление

### Кнопки
| Действие | Левая кнопка (D4) | Правая кнопка (D5) |
|----------|-------------------|-------------------|
| Короткое нажатие | Пробуждение дисплея | Переключение слайда |
| Длинное нажатие (>0.8 сек) | Полив (если влажность ≤75% и нет блокировки) | Переключение AUTO/MANUAL |

### Автоматика
- При влажности **ниже 80%** в заданное время (12:00, 17:00, 20:00) запускается полив
- После полива проверяется поднятие влажности. Если влажность осталась **ниже 25%** — счётчик неудач увеличивается
- После **3 неудачных поливов подряд** система блокируется и выводит `NO WATER!` на LCD и в веб-интерфейс
- Сброс блокировки — кнопка в веб-интерфейсе

### Псевдо-спячка
- Дисплей автоматически гаснет через **3 минуты** бездействия
- Любое нажатие левой кнопки пробуждает дисплей

## 🌐 Web-интерфейс

ESP создаёт **точку доступа** с параметрами:

- **SSID:** `ESP_Watering`
- **Пароль:** `12345678`
- **IP-адрес:** `192.168.4.1`

### Функции веб-интерфейса
- Отображение текущей влажности
- Отображение времени и даты
- Переключение режима AUTO/MANUAL
- Ручной полив
- Установка даты и времени (вручную или синхронизация с телефоном)
- Настройка расписания автополива (три времени)
- Сброс ошибки "НЕТ ВОДЫ"
- Автоматическое обновление данных каждые 2 секунды

## ⚙️ Особенности работы реле

**Почему реле управляется через `INPUT`?**

При включении реле происходит кратковременная просадка питания из-за скачка тока. Это вызывает сбой I2C и глюки дисплея. Решение:

1. Перед включением реле дисплей выключается (`lcd.noBacklight()`)
2. Делается пауза 100 мс
3. Реле включается (`pinMode(OUTPUT); digitalWrite(LOW)`)
4. Через 5 секунд реле выключается (`pinMode(INPUT)`)
5. Пауза 200 мс
6. Дисплей включается (`lcd.backlight()`)

Такой подход полностью исключает глюки дисплея при работе реле.

## 🕐 Настройка времени

ESP хранит время с учётом часового пояса **MSK (UTC+3)**. Синхронизация возможна двумя способами:

1. **Автоматически:** нажать "СИНХРОНИЗИРОВАТЬ С ТЕЛЕФОНОМ" — ESP получит текущее время с вашего устройства
2. **Вручную:** выбрать дату и время в поле "УСТАНОВИТЬ ДАТУ И ВРЕМЯ" и нажать кнопку

При старте установлено **26 марта 2026 года, 00:00 MSK (Четверг)**.

## 📦 Загрузка кода

### Требования
- **Arduino IDE** (версия 1.8.19 или новее) или **PlatformIO**
- Плата: **NodeMCU 1.0 (ESP-12E Module)**
- Библиотеки:
  - `Wire` (встроенная)
  - `LiquidCrystal I2C` by Frank de Brabander
  - `ESP8266WiFi` (встроенная)
  - `ESP8266WebServer` (встроенная)
  - `time.h` (встроенная)

### Установка библиотек
В Arduino IDE: **Инструменты → Управление библиотеками...** → установить `LiquidCrystal I2C`.

### Настройка платы
- **Плата:** NodeMCU 1.0 (ESP-12E Module)
- **CPU Frequency:** 80 MHz
- **Flash Size:** 4M (1M SPIFFS)
- **Debug Port:** Disabled
- **Debug Level:** Нет
- **Upload Speed:** 921600

### Компиляция и загрузка
1. Открыть файл `.ino` в Arduino IDE
2. Подключить ESP8266 к компьютеру через USB
3. Выбрать порт в **Инструменты → Порт**
4. Нажать **Загрузка**

## 🧪 Тестирование

После загрузки прошивки:

1. ESP создаст Wi-Fi сеть `ESP_Watering` (пароль `12345678`)
2. Дисплей **не светится** — система в режиме сна
3. Нажмите **левую кнопку** — дисплей включится, покажет время и влажность
4. Подключитесь к Wi-Fi `ESP_Watering` с телефона
5. Откройте браузер по адресу `http://192.168.4.1` — откроется веб-интерфейс

## 🔄 Дополнительная информация

### Формула расчёта влажности
Датчик выдаёт значение от 340 (полностью мокрый) до 1024 (абсолютно сухой). Процент влажности рассчитывается по формуле:
```влажность = (1024 - показания) * 100 / (1024 - 340)```

### Полив при высокой влажности
Ручной полив блокируется при влажности **выше 75%** — кнопка в веб-интерфейсе становится недоступной, на LCD выводится сообщение.

### Автополив
Срабатывает только в режиме **AUTO**, при влажности **ниже 80%** и в заданное время (12:00, 17:00, 20:00).

### Защита от частого полива
Полив в один и тот же день выполняется только один раз для каждого расписания — если автополив сработал в 12:00, в 17:00 и 20:00 того же дня повторно не включится.
